Finding MIN MAX from an array

Problem Statement 02:

Write a program in any language to find the maximum and minimum element in an array.

Sample Input :
5
48 5 12 95 8 10

Sample Output :
Min: 5
Max: 95


Sample Code in C: 
                
                #include < bits/stdc++.h >
                using namespace std ;
        
                int main(){
                    int n;
                    cin >> n ;
                    vector < int > v(n);
                    for(int i=0 ; i < n ; i++)
                        cin >> v[i];
                    int mx = v[0] , mn = v[0];
                    for(int i=0 ; i < n ; i++){
                        mx = max(mx , v[i]);
                        mn = min(mn , v[i]);
                    }
                    cout<<"Min: %d",min<<\n;
                    cout<<"Max: %d",max<<\n;
                    return 0;
                }


variable Used :
int v,i,n,mn,mx are the variable used for these code

Time Complexity :
O(n)

Space Complexity :
O(1)
